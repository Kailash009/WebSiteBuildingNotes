>> Tim Berner's Lee.
>> It is used to describe the content of a webpage.
>> Html is not a case sensitive.
>> Html is all about tags and attributes.
----------------------------------------------------------
Tags in HTML:-
-----------------------
There are two types of tag in HTML:-
1. Paired/Container Tag. -> which is open and close.
<body></body>
<p></p> 
<div></div> etc.
2. Unpaired/Empty Tag. -> which open and close itself.
For Example:- <img/> 
<br/> -> TO change the line.
<hr/> -> Horizontal Ruler.

Frame:-
---------------
>> Frame is used to divide the content of your webpages.
-------------------------------------------------------------------
FrameSet:-
----------------
>> Frameset is a collection of Frames.
>> Frameset doesn't use body tag.[It doesn't render body].
>> Frameset used rows and cols attributes to divide the content of your webpages.
----------------------------------------------------------------------------------
Syntax:-
---------------
<frameset rows="" cols="">
</frameset>
